How does it work? It's simple!

Before you make an alert message for real, you should try to

There's a 300-characters limit on the textbox though.. Don't worry,

Alert is some kind of text box that pops up when a website tries to tell you something!

You can write an alert message using a JavaScript.

The code to show an alert message is by writing alert("*Your message here*"); in your computer.
